📊 GhReadmeStats

Personalized GitHub Readme Stats API — self-hosted, customizable, accessible from Iran

🇬🇧 English 🇮🇷 فارسی

🇬🇧 API Endpoints

GET /api/stats?username=USER&theme=dark
GitHub stats card with rank visualization
GET /api/top-langs?username=USER&theme=dark
Most used languages card with progress bars

Try it

Parameters

username   — GitHub username (required)
theme      — default, dark, radical, merko, tokyonight, dracula, nord, and 50+ more
hide       — comma-separated: stars,commits,prs,issues,repos,contribs
hide_rank  — true | false
hide_border — true | false
show_icons — true | false
title_color, text_color, icon_color, bg_color, border_color, ring_color — hex (no #)
