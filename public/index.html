GhReadmeStats

Personalized GitHub Readme Stats API

Usage

![Stats](https://gh-readme-stats.pages.dev/api/stats?username=yourname&theme=dark)
![Top Languages](https://gh-readme-stats.pages.dev/api/top-langs?username=yourname&theme=dark)

Parameters

  • username - GitHub username (required)
  • theme - dark, light, radical, merko, tokyonight, synthwave, dracula, nord
  • hide - comma-separated list: stars,commits,prs,issues,repos,contribs
  • hide_rank - true/false
  • show_icons - true/false
  • include_all_commits - true/false
  • bg_c, title_c, text_c, icon_c, etc. - custom colors
